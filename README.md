# ⚗️ Laboratorio de Alquimia - Fluid Physics Simulator

A beautiful, interactive web application featuring:

## 📦 Projects Included

### 1. **Fluid Physics Laboratory** (index.html)
An interactive neuro-aesthetic fluid simulator with real-time physics.

**Features:**
- 🌊 Real-time fluid simulation using Canvas API
- 🎨 Neon aesthetic with glowing particle effects
- 🧪 6 Alchemical elements: Hydrogen, Oxygen, Mercury, Aceite, Water, Plasma
- 📊 Particle interactions and realistic physics
- 🎮 Drag-and-drop interface for element manipulation
- 📈 Real-time statistics (Energy, Temperature, Particle count)

**How to Use:**
1. Open `index.html` in your browser
2. Move the mouse to create waves
3. Drag elements from the sidebar to the canvas
4. Observe realistic fluid behavior (oil repels water, etc.)
5. Adjust viscosity and speed controls

---

### 2. **Weather Dashboard** (weather-dashboard.html)
A comprehensive weather information dashboard with real-time data.

**Features:**
- 🌍 Search weather by city name
- 📍 Geolocation support
- 🌡️ Current weather with detailed metrics
- 📅 5-day forecast
- ⏰ 24-hour hourly forecast
- 💨 Air quality index and pollutant levels
- 💾 Recent search history (localStorage)
- 📱 Fully responsive design

**API Used:**
- OpenWeatherMap API (free tier)
- Get your free API key: https://openweathermap.org/api

**Setup:**
1. Get a free API key from OpenWeatherMap
2. Replace `'a6d9e1b9e7c1d5f3a8e2c4g6h9j1k3m5'` in `weather-script.js` with your key
3. Open `weather-dashboard.html` in your browser
4. Search for any city to see weather data
